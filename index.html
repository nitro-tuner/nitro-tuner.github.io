Overview

Nitro is a programmer-directed code variant tuning framework, jointly developed by the University of Utah and NVIDIA Research.

Nitro utilizes machine learning-based classification to automatically find the best implementation (variant) of a computation for a given input. It provides C++ and Python interfaces for programmers to specify variants, input dataset features, and constraints. In 5 real-world irregular GPU benchmarks, Nitro-tuned variants achieve over 93% of the performance achieved by variants selected through exhaustive search on new, unseen inputs.

Getting Started

You can download a pre-alpha version of Nitro from here. Alternatively, feel free to clone from the Github repository directly. For installation instructions and getting started, check out the tutorial.
