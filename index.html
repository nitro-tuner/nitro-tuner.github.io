Nitro

Automatic Performance Tuning Framework

Overview

Nitro is a programmer-directed code variant tuning framework, jointly developed by the University of Utah and NVIDIA Research.

Nitro provides a library interface that permits programmers to express code variants (alternative implementations of a computation) along with meta-information that aids the system in selecting among the set of variants at run time.

Machine learning is employed to build a model through training on this meta-information, so that when a new input is presented, Nitro can consult the model to select the appropriate variant.
